# cesium-item
基于@vue/cli 4.5.7+cesium1.82的Cesium开发模板
## 项目安装、启动、打包

```
npm install

```

```
npm run serve
```

```
npm run build
```

### 申请并配置AccessToken(可选)

1.登录到("https://cesium.com/ion")注册一个免费的Ion账户；
2.设置属于自己的AccessToken。
